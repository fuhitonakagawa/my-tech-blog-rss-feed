Rust Blog
The Rust Security Response Team was notified that Cargo incorrectly normalizedthe URLs of third-party registries using the sparse index protocol. If ahosting provider allowed multiple registries to be hosted with arbitrary nameswithin the same domain, an attacker able to publish crates in a registry couldobtain the credentials of others users of the same registry.This vulnerability is tracked as CVE-2026-5222.
